# 元图后端服务云平台 - Mmbs

Metamap Backend Server(Mmbs)专注于为前端应用提供一整套后端云服务
无需再造应用后端服务,轻松拥有开发中需要的各种后端能力
为开发者提供数据存储、数据查询、权限控制、文件服务、实时数据同步、云端代码逻辑等开发整体解决方案。
